# HTML Resume Project

A simple, semantic, and responsive **HTML Resume** built using pure HTML5.  
This project demonstrates correct usage of semantic tags, structured content, and clean markup suitable for beginners and evaluations.

---

## Features

- Semantic HTML5 structure (`header`, `nav`, `section`, `article`, `footer`)
- Well-organized resume layout
- Tables for Experience and Education
- Lists for Skills and Projects
- Beginner-friendly and easy to understand
- No external libraries required

---

## Technologies Used

- **HTML5**
- Basic browser styling (no CSS framework)
